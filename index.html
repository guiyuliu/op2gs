OP2GS: Object-Aware 3D Gaussian Primitives via Dual Opacity

Compact and efficient open-vocabulary 3D segmentation with dual-opacity Gaussian primitives.

1University of Oulu 2Aalto University

OP2GS decouples visual opacity from instance occupancy, suppressing mislabeled floater Gaussians while preserving photorealistic rendering.

Abstract

3D Gaussian Splatting provides an explicit and efficient scene representation, but its primitives lack inherent object-level identity. Existing open-vocabulary methods either distill high-dimensional features into Gaussians or lift 2D mask labels into 3D, which can be expensive or vulnerable to label contamination.

We propose OP2GS, an object-aware Gaussian representation that augments each primitive with an explicit instance identity and a dedicated instance opacity, sigma*, for object-mask rendering. The original opacity remains responsible for visual reconstruction, while sigma* models whether a Gaussian should contribute to a particular object mask.

This dual-opacity formulation decouples visual existence from instance occupancy: mislabeled Gaussians can remain available for image rendering while becoming transparent in the object-mask branch. Object-level CLIP descriptors are then aggregated from multi-view rendered masks, avoiding per-Gaussian feature storage.

Method Overview

OP2GS first trains standard 3DGS, initializes scalar object labels from SAM2 masks with multi-view voting, then learns an additional instance opacity using a visibility-aware random object loss. After training, each object receives an aggregated open-vocabulary descriptor from masked multi-view crops.
